LibraryManagementSystem

This program is like a small computer system that works like a real library.
It helps people borrow, return, and search for items such as books, DVDs, and magazines. There are three types of users: students, teachers, and admins and each type of user has different rules. For example, students can only borrow a few handful of books, teachers can borrow any items, but has a limit of borrowing, and admins manage the system, but do not borrow items.

The system keeps track of every item and its status, such as whether it is available in the library, already borrowed, or lost. It also makes sure that users follow the rules by checking borrow and return restrictions for each users and everything before they borrow or return items.

Users can search for items by typing the keyword or the title, and the system will show matching results. The admin can also see reports of all borrowed, available, and lost items. Also, the system can save all the data into the files so we don't loose any data when the program stops.
